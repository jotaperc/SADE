# SADE: Sistema de Analítica de Datos Educativos

SADE es una herramienta de **Data Analytics** diseñada para transformar el registro tradicional de calificaciones en insumos estratégicos para la gestión curricular.

### 🧠 El Enfoque (Pedagogía + Datos)
Como Magíster en Educación, diseñé esta herramienta para resolver la brecha entre la recolección de notas y la toma de decisiones pedagógicas. El sistema procesa puntajes brutos y los segmenta por **Habilidades Cognitivas**.
